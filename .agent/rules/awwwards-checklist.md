# Awwwards Checklist

> Critères pour atteindre le niveau Awwwards

---

## CRITÈRES DE NOTATION

| Critère        | Description                        | Objectif |
| -------------- | ---------------------------------- | -------- |
| **Design**     | Esthétique, cohérence, originalité | 9/10     |
| **Usability**  | UX, navigation, accessibilité      | 8/10     |
| **Creativity** | Innovation, interactions uniques   | 9/10     |
| **Content**    | Qualité, storytelling              | 8/10     |

---

## DESIGN

### Layout & Composition

- [ ] Grille visible et cohérente
- [ ] Alignements parfaits
- [ ] Hiérarchie visuelle claire
- [ ] Espace blanc généreux
- [ ] Asymétrie maîtrisée

### Typographie

- [ ] Font distinctive (Inter)
- [ ] Échelle contrastée (96px → 16px)
- [ ] Line-height optimisé
- [ ] Letter-spacing ajusté sur headlines
- [ ] Max 2 fonts

### Couleurs

- [ ] Palette limitée (2-3 couleurs)
- [ ] Contraste WCAG AA
- [ ] Cohérence light/dark mode

### Détails

- [ ] Border-radius cohérent
- [ ] Shadows subtiles
- [ ] Grain/noise texture
- [ ] Transitions sur tous les états

---

## ANIMATIONS & INTERACTIONS

### Scroll

- [ ] Smooth scroll (Lenis)
- [ ] Reveal animations sur tous les éléments
- [ ] Parallax sur 2-3 sections
- [ ] Timing cohérent

### Hover States

- [ ] Tous éléments cliquables ont un hover
- [ ] Transitions smooth (min 0.2s)
- [ ] Feedback visuel clair

### Micro-interactions

- [ ] Boutons avec effet (scale, shadow)
- [ ] Links avec underline animé
- [ ] Cards avec lift effect
- [ ] Inputs avec focus élaboré

### Avancé

- [ ] Loader/intro au chargement
- [ ] Page transitions
- [ ] Split text sur headlines
- [ ] Stagger sur listes
- [ ] 1 animation "wow" unique

### 3D / WebGL

- [ ] Au moins 1 élément 3D interactif
- [ ] Réagit au scroll ET/OU souris
- [ ] 60fps
- [ ] Fallback mobile

### Custom Cursor

- [ ] Cursor personnalisé
- [ ] Réagit aux éléments
- [ ] Smooth follow
- [ ] Désactivé sur mobile

---

## USABILITY & UX

### Navigation

- [ ] Navigation claire
- [ ] État actif visible
- [ ] Retour en haut facile
- [ ] Menu mobile élégant

### Performance

- [ ] Lighthouse > 90
- [ ] FCP < 1.5s
- [ ] TTI < 3s
- [ ] CLS < 0.1
- [ ] Images optimisées

### Responsive

- [ ] Mobile-first
- [ ] Touch targets > 44px
- [ ] Pas de scroll horizontal
- [ ] Typo lisible mobile

### Accessibilité

- [ ] Contraste WCAG AA
- [ ] Navigation clavier
- [ ] Focus states visibles
- [ ] Alt text images
- [ ] prefers-reduced-motion
- [ ] Semantic HTML

---

## CONTENT

### Copywriting

- [ ] Headlines percutants
- [ ] Pas de lorem ipsum
- [ ] Ton cohérent
- [ ] CTA clairs

### Visuels

- [ ] Images haute qualité
- [ ] Style cohérent
- [ ] Mockups professionnels
- [ ] Icônes custom

---

## POINTS BONUS

| Effet                           | Impact    |
| ------------------------------- | --------- |
| Cursor blob qui change de forme | Fort      |
| 3D morph entre sections         | Très fort |
| Text reveal cinématique         | Fort      |
| Distortion shader sur images    | Fort      |
| Scroll horizontal narratif      | Fort      |

---

## ANTI-PATTERNS

❌ Animations > 1s pour un reveal
❌ Animations qui bloquent la lecture
❌ Trop d'effets différents
❌ Performance < 60fps
❌ Scroll hijacking agressif
❌ 3D qui rame
❌ Mobile bâclé
❌ Dark mode cassé
❌ Boutons sans hover
❌ Loading > 3s sans feedback

---

## PRÉ-SOUMISSION

### Technique

- [ ] Site en ligne (HTTPS)
- [ ] Pas de console errors
- [ ] Lighthouse > 85
- [ ] Testé Chrome, Safari, Firefox
- [ ] Testé mobile réel

### Contenu

- [ ] Contenu final
- [ ] Pas de placeholder
- [ ] Mentions légales
